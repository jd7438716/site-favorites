# Chrome Web Store 上架清单（建议）

## 你需要准备的素材/文件

- 扩展包（zip）：至少包含 `manifest.json`、`contentScript.js`
- 图标（必备）：PNG 格式
  - `16x16`、`48x48`、`128x128`（用于扩展与商店展示）
- 商店截图：建议 3–5 张
  - 展示右侧悬浮按钮、打开侧边栏后的列表、搜索/编辑/拖拽添加等
- 详细描述文案（可直接复用下方「推荐文案」）
- 隐私政策：可使用本仓库的 `PRIVACY.md`，并在商店后台填写/托管
- 支持/联系信息：邮箱或 GitHub 仓库地址

## 可选但强烈建议

- `LICENSE`：开源/闭源都建议明确
- `CHANGELOG.md`：发布版本记录

## 推荐文案（可用于商店描述）

这是一款「域名级收藏夹」扩展：每个网站都有自己的收藏列表。

适合两类场景：

- 网站本身没有收藏夹功能
- 不想/不能登录账号，但仍希望快速保存该站点常用入口

主要功能：

- 右侧悬浮按钮，一键打开站点侧边栏
- 一键收藏/取消收藏当前页面
- 站点内收藏列表：搜索、编辑标题/URL、删除
- 拖拽链接到列表区域，快速添加

隐私说明：

- 数据仅保存在本机 `chrome.storage.local`
- 不上传数据、不做追踪、不需要登录
